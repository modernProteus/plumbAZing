How this works

  1. The Google Sheet is the source of truth. This console is a helper -- it never edits the sheet by itself.
  2. Build a row (or several) below, then copy and paste into the matching tab.
  3. Click Publish > Publish to site in the sheet's menu. The live site rebuilds in a few minutes.
  4. Fail-safe by design: a bad row or a down endpoint can never blank the live site -- it just keeps showing the last good version.
