How this works
  1. Build a promo with the tool above and click Copy row.
  2. Paste it as a new row in the Promotions tab.
  3. Set active to checked and give it a start/end date if it is time-limited.
  4. Click Publish > Publish to site. The site rebuilds and updates in a few minutes.

A promo only shows when it is active AND today is inside its date window. Blank start = live now. Blank end = runs until you turn it off.

This console is read-only: it never writes to the sheet or the site by itself. Nothing here can take the site down.
